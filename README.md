# Mealz

This is a random meal picker.[See it here.](https://mealz.netlify.app/)

## Development

First install dependencies:

```sh
yarn
```

To run in hot module reloading mode:

```sh
yarn serve
```

To lint all files:

```sh
yarn lint
```

## Production

To create a production build:

```sh
yarn build
```

## Running

Open `dist/index.html` in your browser.
